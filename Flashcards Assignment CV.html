What is the purpose of the "head" element?
------------------------------------------------------------------------- It contains metadata about the document, such as the title, links to CSS stylesheets, and information for search engines.
What is the purpose of the "body" element?
------------------------------------------------------------------------- It contains the visible content of the web page that users see and interact with.
How do you create a heading in HTML?
------------------------------------------------------------------------- Use heading tags "h1" to "h6"with "h1" being the largest and most important heading.
How do you create a paragraph in HTML?
------------------------------------------------------------------------- Use the "p" tag.
How do you create a link in HTML?
------------------------------------------------------------------------- Use the "a" tag with the href attribute to specify the link's destination.
How do you insert an image in HTML?
------------------------------------------------------------------------- Use the "img"tag with the src attribute to specify the image's source file.
How do you create a list in HTML?
------------------------------------------------------------------------- Use either an unordered list "ul" for bullet points or an ordered list "ol" for numbered items.
Which HTML tag adds a bold line of text?
------------------------------------------------------------------------- "strong"
